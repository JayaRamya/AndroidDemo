Simple Calendar

Introduction

This project is just a try to provide overview of Calendar. I tried to include almost five types of color styles with each having different Background, Current Day, Selected Day, and Event Marker.

As currently, I did not provide support for Event creation but I have included functionality for that.

Code Snippet

Rather than using Background Image, in this project, I tried to use GradientDrawable. In this demo, I have included abhan, Red, Blue, Black, Green, and Orange colors for integrated styles.

Now get Calendar instance and set current date, month, and year.

calMonth = Calendar.getInstance();
calMonth.set(calMonth.get(Calendar.YEAR), calMonth.get(Calendar.MONTH),
	calMonth.get(Calendar.DATE));

Now initialize Adapter with current set Calendar instance.

adapter = new CalendarAdapter(CalendarActivity.this, calMonth,
	parentGridBackgroundColor, currentMonthDateColor,
	prevNextMonthDateColor, smallCircle,

	sundayColor);

Create a method which refresh view according to selected month and year.

	public void refreshDays() {
		lastClicked = -1;
		days = null;
		firstDay = (short) month.get(Calendar.DAY_OF_WEEK);
		if (firstDay == 7) {
			days = new String[42];
		} else {
			days = new String[35];
		}
		short j = FIRST_DAY_OF_WEEK;

		if (firstDay > 1) {
			Calendar calendar = Calendar.getInstance();
			if (month.get(Calendar.MONTH) == Calendar.JANUARY) {
				calendar.set(month.get(Calendar.YEAR) - 1, Calendar.DECEMBER, 1);
			} else {
				calendar.set(month.get(Calendar.YEAR),
						month.get(Calendar.MONTH) - 1, 1);
			}
			short length = 0;

			for (j = 0; j < firstDay - FIRST_DAY_OF_WEEK; j++) {
				days[j] = "";
				length = j;
			}

			short maxDayPrevious = (short) calendar
					.getActualMaximum(Calendar.DAY_OF_MONTH);

			for (int i = length - 1; i >= 0; i--) {
				days[i] = String.valueOf(maxDayPrevious);
				maxDayPrevious--;
			}
		} else {
			for (j = 0; j < FIRST_DAY_OF_WEEK * 6; j++) {
				days[j] = "";
			}
			j = FIRST_DAY_OF_WEEK * 6 + 1;
		}

		short dayNumber = 1, startDay = 1;
		maxday = (short) month.getActualMaximum(Calendar.DAY_OF_MONTH);
		for (short i = (short) (j - 1); i < days.length; i++) {
			if (dayNumber <= maxday) {
				days[i] = "" + dayNumber;
				dayNumber++;
			} else {
				days[i] = "" + startDay;
				startDay++;
			}
		}
	}

To move next or previous month, you can use Left or Right arrow.
