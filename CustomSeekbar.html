Custom Seekbar

Introduction

I developed this demo, when Android Ice Cream Sandwich OS launched, to provide common GUI like ICS in all pre HoneyComb devices. Now-a-days many web-sites provide graphics and styles to create kind of UI.

Code Snippet

Simply take Images from Android SDK platform directory and put it into your respective drawable-**** folder. Path would be something like this ..\AndroidSDK\platforms\android-14\data\res\drawable-mdpi

Images to get from this folder...

scrubber_control_disabled_holo.png
scrubber_control_focused_holo.png
scrubber_control_normal_holo.png
scrubber_control_pressed_holo.png
scrubber_primary_holo.9.png
scrubber_secondary_holo.9.png
scrubber_track_holo_dark.9.png
scrubber_track_holo_light.9.png

Create a seekbar_progress.xml, seekbar_progress_bg.xml, and seekthumb_selector and put into drawable folder.

Set seekbar_progress.xml into android:progressDrawable attribute and set seekthumb_selector into android:thumb attribute as shown below.

android:max="100"
android:progress="0"
android:progressDrawable="@drawable/seekbar_progress"
android:secondaryProgress="0"
android:thumb="@drawable/seekthumb_selector"

Custome Seekbar with provided drawable is ready to use.
